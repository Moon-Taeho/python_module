## 練習問題②
- 変数aの値が3で割り切れるときには「3でわりきれます」、そうでないときには「3で割り切れません」とコンソールに出力するプログラムを完成させてください。
``` python
a = 2021
if a % 3 == 0:
    print("3で割り切れます")
else:
    print(a % 3)
```
